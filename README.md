# Dentate Gyrus Granule Cell Multi-compartment Model created for BI4001 Advanced Research Project. 

To run the dissertation files, clone the repository, then compile mod files using nrnivmodl on linux/mac, or visit the NEURON doccumentation for windows instructions. 
https://www.neuronsimulator.org/en/latest/guide/index.html 

Required packages and versions are stated at the top of the scripts. 

Check the directory paths at the top of the Submission_Current_Clamp.py and Submission_Voltage_Clamp.py scripts to make sure they are correct for your set up. Scripts contain error warnings which should explain the issue if the code does not run e.g., if the mod files are not found because the patgh is incorrect. 

please note, the script must be in the same firectory as the ic and vc raw data outputs to run properly. if no, either move the script or set the paths in the r script before running.

The three cell conditions are made within the scripts and are identical between the scripts. 

Axes for plots may differ depending on how you set them to run i.e., decimal or standard form, same or unique etc. Default is the autogenerated axes. 


Archieve contains prior versions and different implementations, which are not used for the dissertation...they are very messy but follow the concept of 'building in public'.
